Платформа для управления цифровыми картами лояльности в Apple Wallet и Android-кошельках

Срок реализации 5–6 месяцев

📋 Оценка на основе технического задания

⚙️ Модуль 1 · Ядро и эмиссия

  • Генерация .pkpass для Apple Wallet и JSON для Android
  • Поддержка скидочных, бонусных, депозитных, подарочных, штамп-карт
  • Привязка существующих пластиковых карт
  • Безлимитный выпуск, работа с push-токенами
≈ 1 200 000 ₽

🎨 Модуль 2 · Дизайн и кастомизация

  • Визуальный редактор (drag-n-drop, загрузка фонов, шрифты)
  • Динамические шаблоны под уровни лояльности
  • Настройка отображаемых полей (баллы, статус, реферальный код)
≈ 800 000 ₽

🔌 Модуль 3 · Интеграционный слой

  • RESTful API для полного управления картами
  • Коннектор к iiko (синхронизация гостей, баллов, штрихкодов)
  • Webhooks для событий (регистрация, обновление)
≈ 1 200 000 ₽

📲 Модуль 4 · Онбординг клиентов

  • Конструктор анкет (ссылка / QR-код)
  • Автоматическая выдача карты после заполнения
  • Передача данных в CRM (имя, телефон, email, дата рождения)
≈ 600 000 ₽

📣 Модуль 5 · Маркетинговые коммуникации

  • Push-уведомления (транзакционные, маркетинговые, гео)
  • Динамическое обновление данных на карте (до 10+ раз/час)
  • Триггеры по геозонам (при входе в магазин)
≈ 800 000 ₽

👤 Модуль 6 · Личный кабинет клиента

  • Веб-версия для просмотра баланса, истории, статуса
  • Резервный доступ без Wallet (Android / десктоп)
